隐私政策

我们重视并保护您的隐私

1. 信息收集

Photo Monster 是一款客户端摄影辅助工具，我们尽可能减少数据的收集和使用：

  • 照片数据：您上传用于分析的照片仅在浏览器本地处理，不会上传到任何服务器。照片的 EXIF 数据在本地读取后用于分析，不会被存储或传输。
  • AI 配置信息：您在 AI 配置页面填写的 API 密钥等敏感信息仅保存在浏览器的 localStorage 中，不会发送到任何第三方服务器。
  • 使用偏好：您的主题偏好、浏览记录等非敏感信息可能保存在本地存储中，以提升使用体验。

2. 数据存储

所有数据存储均在您的浏览器本地完成：

  • 使用浏览器 localStorage 存储配置和偏好
  • 使用 Service Worker 实现离线缓存，缓存内容仅为静态资源
  • 我们不使用 Cookie 追踪用户行为
  • 我们不收集任何可识别个人身份的信息

3. 第三方服务

本应用可能使用以下第三方服务：

  • CDN 服务：通过 CDN 加载前端库（如 Font Awesome、Google Fonts），这些服务可能记录标准的访问日志。
  • AI 分析服务：如果您配置了通义千问等 AI 服务，您的分析请求将直接发送到对应的 AI 服务提供商，受其隐私政策约束。

4. 数据安全

我们采取以下措施保护您的数据安全：

  • 所有敏感操作（照片分析、AI 交互）均在浏览器本地完成
  • API 密钥等凭证仅存储在本地，不经过我们的服务器
  • 使用 HTTPS 协议加载外部资源

5. 用户权利

您随时可以：

  • 清除浏览器缓存和 localStorage 数据
  • 在浏览器设置中卸载 Service Worker
  • 停止使用本应用

6. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，建议您定期查看。

7. 联系方式

如有任何隐私相关问题，请通过 GitHub 仓库提交 Issue。

8. 用户上传内容（照片墙功能）

当您使用照片墙功能上传内容时，以下隐私条款适用：

  • 内容存储：您选择上传至照片墙的照片将存储于服务器，并对其他用户公开展示。上传前请确保您拥有该内容的合法权益。
  • EXIF 数据：上传的照片可能包含 EXIF 元数据（如拍摄地点、设备信息）。建议上传敏感内容前自行清除 EXIF 数据，或授权我们自动清除。
  • 删除权：您可随时删除自己上传的内容，删除后将立即从公开展示中移除。已删除内容可能在系统备份中保留不超过30天后彻底清除。
  • 数据使用：我们不会将您的上传内容用于训练 AI 模型或提供给第三方，除非获得您的明确授权。
